1. Напишите функцию, которая принимает на вход текст на русском языке и возвращает количество гласных букв в нём.

2. Напишите функцию, которая принимает на вход текст и возвращает самое длинное слово (последовательность символов от пробела до пробела).

3. Напишите функцию, которая будет определять, является ли введённая строка палиндромом. Пробелы и регистр не должны учитываться.

4. Напишите функцию, которая принимает на вход предложение и возвращает его же с обратным порядком слов.

5. Напишите программу, которая будет считать манулов.
